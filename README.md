# Binance Futures Testnet Trading Bot

## Features

- Market Orders
- Limit Orders
- BUY and SELL Support
- Logging
- Validation
- Error Handling

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python cli.py
```

## Project Structure

bot/
- client.py
- orders.py
- validators.py
- logging_config.py
- cli.py
